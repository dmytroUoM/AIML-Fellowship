Created one combined CPU/GPU NVIDIA Real-ESRGAN script because the code difference is small. It automatically uses an NVIDIA GPU if CUDA is available; otherwise, it falls back to CPU. I also added commented lines so you can force CPU or GPU if needed.

Scientific tomography note set OUTSCALE = 2
rather than full 4, because for scientific resistance-tomography frames, x2 is usually safer visually. Full AI upscaling can make images look better, but it may also invent artificial detail. Keep your original Frames/Cleaned images as the scientific source data, and use Improved_Real-ESRGAN mainly for presentation/visualisation unless validated.
