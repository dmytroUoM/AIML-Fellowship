Approach: the numbering is a distinct colour (magenta, ~R255/B255/G-varies) that never appears in the plot itself, so this is a colour-key + imprint problem rather than an OCR/detection problem — much simpler and more robust.

Colour mask — flag pixels where R and B are both high and G is comparatively low (catches the solid magenta core and the anti-aliased pink edges that fade toward white).
Dilate the mask by a couple of pixels so no faint pink fringing survives.

Imprint (cv2.inpaint, Telea algorithm) the masked pixels with the surrounding image content, so the removed text is filled in seamlessly rather than leaving white gaps.
Save the result (and a debug mask image, useful if you need to tune thresholds).

Ran a command, read a file. The script is generic — run python3 remove_numbering.py <input> <output> on any similar frame, and if you have a whole batch (e.g. frame_0001.png, frame_0002.png, ...) it's a one-line loop to apply it to all of them. If a future frame uses a different label colour, just tweak the thresholds in build_magenta_mask.
